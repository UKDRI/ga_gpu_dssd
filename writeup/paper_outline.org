* Abstract
- Background :: Parkinson's Disease (PD) is characterized by the progressive loss of dopaminergic neurons, yet these cells do not fail in a uniform manner. Single-cell RNA sequencing (scRNA-seq) reveals high heterogeneity, but identifying robust, combinatorial gene signatures that define *distinct neuronal failure modes* remains a computational challenge.
- Methodology :: We applied Diverse Subgroup Set Discovery (DSSD) to this problem, modeling neuronal heterogeneity as a collection of distinct rules. To navigate the vast search space, we developed a *Hierarchical Parallel Genetic Algorithm*. The framework deployed 100 GPU nodes, with each node simultaneously executing 10 independent search instances via tensor batching.
- Rigor :: We strictly enforced an *"Equal Computational Effort"* principle. We performed a massive-scale permutation test where the real dataset and 999 randomized datasets were each subjected to exactly *72 hours* of rapid-cycle evolutionary search (population 55, restarting every 300 generations).
- Results :: The framework executed a deep search equivalent to ~72,000 hours of sequential computation. It identified a combinatorial gene signature in PD dopaminergic neurons with a fitness score that was statistically unreachable by chance ($p < 0.001$). This signature defines a robust, non-overlapping cell state distinct from general stress responses.
- Conclusion :: This study demonstrates that DSSD, powered by parallel GPU computing and rigorous statistics, is a powerful paradigm for untangling the heterogeneity of dopaminergic neurodegeneration.

* Introduction
** The Biological Challenge: The "Mixed Bag" of PD Neurons
- Parkinson's Disease primarily affects dopaminergic neurons in the substantia nigra, but degeneration is asynchronous and heterogeneous.
- *The Core Hypothesis:* "Non-overlapping subgroups" in scRNA-seq data correspond to *distinct neuronal states* or molecular failure modes (e.g., mitochondrial collapse vs. synaptic dysfunction).

** The Computational Challenge
- *The Search Space:* Finding the specific combinations of genes that define these states requires searching an astronomically large solution space ($2^{1000}$ combinations).
- *The Interpretability Gap:* "Black box" AI can classify PD neurons but cannot explain the underlying molecular mechanisms.
- *The Solution:* Diverse Subgroup Set Discovery (DSSD) generates interpretable "IF-THEN" rules, but validating them requires immense compute power to distinguish true biological signal from noise.

** Our Approach: Parallel DSSD on GPUs
- We present a framework that treats identifying PD cell states as a massive combinatorial optimization problem.
- *Parallelism as an Enabler:* We combine the population-based search of Genetic Algorithms with the tensor throughput of GPUs to solve the computational bottleneck.

* Methods
** Single-Cell Data Preprocessing
- *Dataset:* 6,385 dopaminergic neurons from the substantia nigra (Control vs. PD).
- *Normalization:* =SCTransform= with strict batch regression (=Sample_v2=).
- *Input:* The top 1,000 variable genes, binarized to facilitate logical rule generation.

** The Hierarchical GPU-Accelerated Framework
We implemented a two-tier parallel architecture on the DAWN supercomputer (Intel PVC Max Series GPUs).

*** Tier 1: Vectorized Intra-Node Parallelism (The "10x" Kernel)
- Inside each job, we utilized custom PyTorch kernels to run *10 independent Genetic Algorithm searches simultaneously* on a single GPU.
- By using =torch.int8= and 5D tensor broadcasting, we evaluated 10 separate populations (Pop=55) in lockstep.

*** Tier 2: Distributed Inter-Node Parallelism (The "100x" Array)
- We scaled the search across *100 GPU nodes* via SLURM.
- *Task 0:* Processed the Real Data + 9 Randomized Permutations.
- *Tasks 1-99:* Processed 990 Randomized Permutations.
- *Total Scale:* 1,000 datasets (1 Real + 999 Null) evaluated concurrently.

** The Time-Bounded "Rapid-Cycle" Protocol
- *Strategy:* Standard GAs often stagnate in local optima. To maximize exploration, we implemented a *Random Restart* strategy.
- *Micro-Cycle Parameters:*
  - Population Size: 55
  - Run Length: 300 Generations (approx. [X] seconds per burst).
  - Action: At generation 300, the best individual is archived, and the population is fully re-initialized.
- *Macro-Protocol:* This cycle repeats continuously for a strict *72-hour wall-clock limit*, ensuring deep exploration of the landscape for both real and null models.

** Statistical Validation
- *Null Hypothesis:* That the combinatorial rules found in biological neurons are no more fit than patterns found in noise given sufficient search time.
- *The Test:* We compare the global best fitness found in the real PD neurons ($F_{real}$) against the distribution of global bests from 999 randomized datasets ($F_{perm}$), all searched with identical effort.
